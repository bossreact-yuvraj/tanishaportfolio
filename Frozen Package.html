Research Behind

Was wondering to create something out of the box. So thought to blend disney character as a ice cream brand. This was a very challenging project as this type of something was never created. So took reference from popsicle packages, then roughly played with several kind of backgrounds, image placing. Also it was challenging to keep product and character both in focus. To over come this tried different placing of items.

About The Package

This is a package design of ice cream popsicles which are "FROZEN". There are 4 flavors in this which match with the personality of their respective characters. Created package on concept of fantasy. Balanced the focus on both character and popsicle flavors. Also used Frozen's font only to keep the base same. Package colors are of same as the character's outfits.
