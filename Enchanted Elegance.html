Enchanted Elegance

What if Harry Potter sold make up products? This project will give you a journey of Harry Potter inspired make up brand.Enchanted Elegance pays homage to the enchanting world of Harry Potter, infusing the magic and wonder of Hogwarts into every makeup creation.

Research Behind

Before this project thought of making something different than a simple brand. So combined already exisiting movie with some product. And output looks beautiful. While creating this project tried to combine elements of Harry Potter with make up related elements.

About The Logo

For logo used the triangle and the circle which is the icon of Deathly Hollows from the movie, Potterheads know it very well. Inside used ee in cursive which are initials of brand name at the same time there tail depicts magic wand trail and make up brush stroke. The logo is simple, elegant but at the same time it also catches the eyes of potterheads the main target audience of the brand.
