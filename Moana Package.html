Research Behind

After creating project of Frozen got inspired to create another project related to disney package. So now decided to create for Moana. This project came with another level of challenges. For this researched on font styles as the Moana official text was not going with the background so then used a typeface which was in contrast with the logo font as well as which goes well with the theme and background.

About The Package

This package designed is inspired by Disney movie Moana the background text and the theme is based on Moana. How Moana connects to mocktails?? Whenever we think of mocktails we think of beach and island and Moana is set up in an island so that's why Moana suits best with the mocktail brand. Designed 2 flavors sunset related and sunrise related i.e. Island Odyssey.
