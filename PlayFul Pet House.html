Research Behind

For this project wanted to create some unique and playful design of both logo and package so thought to give it a colorful color palette.So for this searched about dogs and pet houses to take some refrence from the original image for Logo and package.

About The Logo & Package

Logo consist of dogs face in shape of P (P are 2 initial letters of our brand name), As well as placed them to create house shape using negative space between these 2 P's, we have used the door as a complimentary icon to beautify and add more accurate meaning to the logo. Used colorful color palette for our brands' logo design. Package is linked with the logo as well as in package tried to create house which makes audience know what actually the brand's product is.
